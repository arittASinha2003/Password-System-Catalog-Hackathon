# Password-System-Catalog-Hackathon

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`USERNAME1`
`PASSWORD1`
`SECURITY_QUESTIONS`
`SECURITY_ANSWERS`

```
USERNAME1 = "XXX"
PASSWORD1 = "YYY"
SECURITY_QUESTIONS="Q1|Q2|Q3"
SECURITY_ANSWERS="A1|A2|A3"
```
